pcontrol

Dieses Verzeichnis enthält einen pvserver, der ein zentrales Event Log und eine Kontrolle Ihrer Hintergrundprozesse ermöglicht.

Gehen Sie in das Verzeichnis und probieren den pvserver aus. Mit "./client" können Sie Event Log Botschaften erzeugen. Sehen Sie sich dazu auch client.cpp an. Sie sehen darin, wie Sie Event Logs in Ihren eigenen Programmen einbauen können.

Lesen Sie auch README.txt in dem Verzeichnis.
